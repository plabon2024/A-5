Question-1: What are the different ways to select an element in the DOM?

There are 5 different way to select element in dom

  1. getElementById
  2. getElementsByClassName
  3. querySelector
  4. querySelectorAll
  5. getElementsByTagName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML content inside an element, including any child elements and tags. innerText: Gets or sets only the visible text inside an element. textContent: Gets or sets all text inside an element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a pattern that allows a parent element to handle events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event triggered on a child element propagates up to its parent elements in the DOM tree.
